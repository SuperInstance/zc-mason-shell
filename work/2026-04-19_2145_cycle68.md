## Edge-Case Tests Written

I created 5 edge-case tests targeting specific boundary conditions and error paths:

### Test 1: Invalid P0 Action with Malformed Constraint
Tests that the room correctly rejects actions claiming to be P0 (negative space mapping) but with malformed constraint data that can't be parsed.

### Test 2: P1 Channel Discovery with Boundary Parameters  
Tests safe channel discovery when parameters are at exact boundary values (minimum/maximum allowed by the deadband).

### Test 3: P2 Optimization with Zero Gradient
Tests optimization path handling when the gradient feedback is zero (no improvement direction available).

### Test 4: Tile Submission with Missing Required Fields
Tests tile validation by submitting tiles missing required metadata fields that the room expects.

### Test 5: State Corruption Recovery
Tests that the room can recover when agent state becomes corrupted (e.g., invalid JSON structure in persisted state).
