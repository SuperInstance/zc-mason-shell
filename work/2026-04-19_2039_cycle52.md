### 1. Test: `test_deadband_room_rejects_p2_without_p1`
**Purpose:** Verify strict P0→P1→P2 priority enforcement.
**Scenario:** Agent attempts to submit a P2 tile before any P1 tile exists in session.
**Expected:** Room rejects tile with `InvalidProtocolOrder` error.
**Code:**
```rust
#[test]
fn test_deadband_room_rejects_p2_without_p1() {
    let mut room = DeadbandRoom::new("test-room".to_string());
    let agent_id = "test-agent".to_string();
    
    // Submit P0 tile (allowed)
    let p0_tile = Tile::new(agent_id.clone(), Priority::P0, "Avoid null pointers".to_string());
    assert!(room.submit_tile(p0_tile).is_ok());
    
    // Attempt P2 tile without P1 (should fail)
    let p2_tile = Tile::new(agent_id, Priority::P2, "Optimize loop".to_string());
    let result = room.submit_tile(p2_tile);
    assert!(result.is_err());
    assert_eq!(result.unwrap_err(), RoomError::InvalidProtocolOrder);
}
```

### 2. Test: `test_deadband_room_handles_concurrent_tile_submission`
**Purpose:** Verify thread safety when multiple agents submit simultaneously.
**Scenario:** Two agents submit P0 tiles at same time using threads.
**Expected:** Both tiles accepted, room state consistent, no data races.
**Code:**
```rust
#[test]
fn test_deadband_room_handles_concurrent_tile_submission() {
    let room = Arc::new(Mutex::new(DeadbandRoom::new("concurrent-room".to_string())));
    let mut handles = vec![];
    
    for i in 0..2 {
        let room_clone = Arc::clone(&room);
        let handle = thread::spawn(move || {
            let agent_id = format!("agent-{}", i);
            let tile = Tile::new(agent_id, Priority::P0, format!("Avoid race condition {}", i));
            let mut room = room_clone.lock().unwrap();
            room.submit_tile(tile)
        });
        handles.push(handle);
    }
    
    for handle in handles {
        assert!(handle.join().unwrap().is_ok());
    }
    
    let room = room.lock().unwrap();
    assert_eq!(room.tile_count(), 2);
}
```

### 3. Test: `test_deadband_room_validates_tile_content_length`
**Purpose:** Verify content length boundaries are enforced.
**Scenario:** Agent submits tile with empty content and extremely long content.
**Expected:** Empty content rejected, reasonable length accepted, extreme length truncated or rejected based on config.
**Code:**
```rust
#[test]
fn test_deadband_room_validates_tile_content_length() {
    let mut room = DeadbandRoom::new("validation-room".to_string());
    let agent_id = "test-agent".to_string();
    
    // Empty content should fail
    let empty_tile = Tile::new(agent_id.clone(), Priority::P0, "".to_string());
    assert!(room.submit_tile(empty_tile).is_err());
    
    // Normal content should pass
    let normal_tile = Tile::new(agent_id.clone(), Priority::P0, "Valid content".to_string());
    assert!(room.submit_tile(normal_tile).is_ok());
    
    // Extremely long content (10KB) - check truncation or rejection
    let long_content = "x".repeat(10240);
    let long_tile = Tile::new(agent_id, Priority::P0, long_content);
    let result = room.submit_tile(long_tile);
    // Either succeeds with truncation or fails with appropriate error
    assert!(result.is_ok() || matches!(result.unwrap_err(), RoomError::InvalidTileContent));
}
```

### 4. Test: `test_deadband_room_persists_state_across_restarts`
**Purpose:** Verify room state survives serialization/deserialization cycles.
**Scenario:** Save room state, create new room from saved state, verify tiles and protocol state intact.
**Expected:** All tiles, agent sessions, and protocol progress preserved.
**Code:**
```rust
#[test]
fn test_deadband_room_persists_state_across_restarts() {
    let mut room1 = DeadbandRoom::new("persistent-room".to_string());
    let agent_id = "test-agent".to_string();
    
    // Submit P0 and P1 tiles
    room1.submit_tile(Tile::new(agent_id.clone(), Priority::P0, "Avoid rocks".to_string()))
        .unwrap();
    room1.submit_tile(Tile::new(agent_id.clone(), Priority::P1, "Find channel".to_string()))
        .unwrap();
    
    // Serialize state
    let state = room1.save_state().unwrap();
    
    // Create new room from state
    let room2 = DeadbandRoom::load_state(state).unwrap();
    
    // Verify same tile count
    assert_eq!(room1.tile_count(), room2.tile_count());
    
    // Verify protocol state allows P2 now
    let p2_tile = Tile::new(agent_id, Priority::P2, "Optimize path".to_string());
    assert!(room2.submit_tile(p2_tile).is_ok());
}
```

### 5. Test: `test_deadband_room_handles_agent_session_timeout`
**Purpose:** Verify stale agent sessions don't block protocol progress.
**Scenario:** Agent starts session with P0, doesn't submit P1 within timeout, new agent submits P0.
**Expected:** New agent can start fresh session, old session data cleaned up or ignored.
**Code:**
```rust
#[test]
fn test_deadband_room_handles_agent_session_timeout() {
    let mut room = DeadbandRoom::new("timeout-room".to_string());
    let agent1 = "agent-1".to_string();
    let agent2 = "agent-2".to_string();
    
    // Agent 1 submits P0
    room.submit_tile(Tile::new(agent1.clone(), Priority::P0, "P0 from agent1".to_string()))
        .unwrap();
    
    // Simulate timeout (manually expire session for test)
    room.expire_agent_session(&agent1);
    
    // Agent 2 should be able to submit P0 (fresh session)
    let result = room.submit_tile(Tile::new(agent2, Priority::P0, "P0 from agent2".to_string()));
    assert!(result.is_ok());
    
    // Agent 1 trying to continue should fail or start new session
    let result = room.submit_tile(Tile::new(agent1, Priority::P1, "Should fail".to_string()));
    assert!(result.is_err()); // Session expired
}
```
