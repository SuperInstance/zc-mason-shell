### Test 1: Concurrent Tile Submission Stress Test
**Scenario:** Simulate 50 agents submitting tiles simultaneously to the same DeadbandRoom instance.
**Edge:** Race conditions, lock contention, state corruption.
**Assertion:** All tiles are processed, room state remains consistent, no data loss.

```python
def test_concurrent_tile_submission(self):
    """P0: Room must handle high concurrency without data corruption."""
    import threading
    room = DeadbandRoom(capacity=1000)
    results = []
    errors = []

    def submit_tile(agent_id):
        tile = {
            "agent_id": f"agent_{agent_id}",
            "cycle": agent_id,
            "content": f"Test tile from agent {agent_id}",
            "protocol_phase": "P1"
        }
        try:
            room.submit_tile(tile)
            results.append(agent_id)
        except Exception as e:
            errors.append((agent_id, str(e)))

    threads = []
    for i in range(50):
        t = threading.Thread(target=submit_tile, args=(i,))
        threads.append(t)
        t.start()

    for t in threads:
        t.join()

    assert len(errors) == 0, f"Errors during concurrent submission: {errors}"
    assert len(results) == 50, f"Only {len(results)} tiles processed, expected 50"
    assert room.tile_count == 50, f"Room tile count mismatch: {room.tile_count}"
```

### Test 2: Network Partition Resilience
**Scenario:** Room server experiences intermittent connectivity loss during tile processing.
**Edge:** Partial writes, orphaned transactions, recovery mechanisms.
**Assertion:** Room rejects tiles gracefully during outage, recovers state post-partition.

```python
def test_network_partition_handling(self):
    """P0: Room must reject tiles when backend services are unavailable."""
    room = DeadbandRoom(capacity=100)
    # Simulate network partition by mocking external service failure
    room._backend_available = False

    tile = {
        "agent_id": "test_agent",
        "cycle": 1,
        "content": "Tile during partition",
        "protocol_phase": "P2"
    }

    with pytest.raises(RoomBackendUnavailableError):
        room.submit_tile(tile)

    # Verify no partial state change
    assert room.tile_count == 0
    assert "test_agent" not in room._agent_sessions

    # Restore connectivity
    room._backend_available = True
    # Should accept tile now
    room.submit_tile(tile)
    assert room.tile_count == 1
```
