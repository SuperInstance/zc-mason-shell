4. Don't test concurrent access (not thread-safe by design).

## Safe Channels (P1)
Where we CAN test:
1. **Boundary values**: `deadband_width` at minimum positive float, `history_size` at 1.
2. **Sequence violations**: P0 > P1 > P2 violation patterns.
3. **History overflow**: More steps than `history_size`.
4. **Reset state**: Verify complete state clearance.
5. **Floating-point precision**: Deadband comparisons with epsilon.

### Test 2: History Circular Buffer Behavior
```python
def test_history_circular_overflow():
    """Test that history acts as circular buffer when exceeding history_size."""
    room = DeadbandRoom(deadband_width=1.0, history_size=3)
    
    # Add 5 steps, should only retain last 3
    for i in range(5):
        room.step(p0=float(i), p1=float(i+1), p2=float(i+2))
    
    assert len(room.history) == 3
    # Should contain steps 2, 3, 4 (indices)
    expected_p0_values = [2.0, 3.0, 4.0]
    actual_p0_values = [entry['p0'] for entry in room.history]
    assert actual_p0_values == expected_p0_values
```

### Test 3: Deadband Boundary Violation
```python
def test_deadband_boundary_exact():
    """Test violation detection when P1 equals P0 + deadband_width (should not violate)."""
    room = DeadbandRoom(deadband_width=2.0, history_size=10)
    
    # P1 at exact boundary: 5.0 + 2.0 = 7.0
    result = room.step(p0=5.0, p1=7.0, p2=8.0)
    
    # Should NOT be a violation (P1 > P0 + deadband_width is required for safety)
    assert not result['violation']
    assert room.violations == 0
```

### Test 4: Reset Clears All State
```python
def test_reset_clears_history_and_violations():
    """Test reset() clears history and violations counter completely."""
    room = DeadbandRoom(deadband_width=1.0, history_size=5)
    
    # Generate some state
    room.step(p0=1.0, p1=0.5, p2=0.1)  # Violation: P1 < P0
    room.step(p0=1.0, p1=3.0, p2=2.0)  # Violation: P2 > P1
    
    assert len(room.history) == 2
    assert room.violations == 2
    
    # Reset
    room.reset()
    
    assert len(room.history) == 0
    assert room.violations == 0
```
