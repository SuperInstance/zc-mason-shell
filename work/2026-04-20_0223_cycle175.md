## Analysis of Existing Tests
The `DeadbandRoom` class implements the fleet's Deadband Protocol (P0: negative space, P1: safe channels, P2: optimize). Existing tests in `test_deadband.py` cover:
- Basic protocol application with valid inputs

- P1 channel identification
- P2 optimization within channels

## Five Additional Edge-Case Tests

### 1. Test: Empty Action Space (P0 Boundary)
**Scenario:** Agent receives an empty list of possible actions.
**Edge:** The protocol should handle this as a P0 violation (no safe channels exist).
**Expected:** Room returns a P0 tile with appropriate error, not crash.
```python
def test_empty_action_space(self):
    """DeadbandRoom should handle empty action list gracefully."""
    room = DeadbandRoom()
    result = room.apply_protocol([], {})
    self.assertIsInstance(result, dict)
    self.assertEqual(result.get('phase'), 'P0')
    self.assertIn('error', result)
```

### 2. Test: All Actions Are P0 Violations
**Scenario:** Every possible action is marked as invalid (negative space).
**Edge:** No P1 channels exist; protocol should not progress to P2.
**Expected:** Room returns P0 result with all actions rejected, no P1/P2 output.
```python
def test_all_actions_p0(self):
    """When all actions are invalid, protocol stops at P0."""
    actions = ['crash_db', 'delete_root', 'format_disk']
    constraints = {'invalid_actions': actions}  # All are P0
    room = DeadbandRoom()
    result = room.apply_protocol(actions, constraints)
    self.assertEqual(result['phase'], 'P0')
    self.assertEqual(len(result.get('safe_channels', [])), 0)
    self.assertNotIn('optimized_path', result)
```

### 3. Test: Single Safe Channel with Multiple Sub-Optimal Paths (P2 Edge)
**Scenario:** Only one P1 channel exists, but it contains multiple possible P2 optimizations.
**Edge:** P2 must still execute even with limited choice.
**Expected:** Room returns P1 channel and performs P2 optimization within it.
```python
def test_single_channel_p2_optimization(self):
    """P2 optimization should run even with only one safe channel."""
    actions = ['write_log', 'read_log', 'delete_log']  # Only 'write_log' is safe
    constraints = {'safe_actions': ['write_log']}
    room = DeadbandRoom()
    result = room.apply_protocol(actions, constraints)
    self.assertEqual(result['phase'], 'P2')
    self.assertEqual(result['safe_channels'], ['write_log'])
    self.assertIn('optimized_path', result)
    # Optimized path should be within the single channel
    self.assertEqual(result['optimized_path'], 'write_log')
```

### 4. Test: Malformed Tile Submission
**Scenario:** Agent attempts to submit a tile missing required fields.
**Edge:** Tile validation should catch missing data before training.
**Expected:** Room rejects tile with specific error, does not add to training set.
```python
def test_malformed_tile_submission(self):
    """DeadbandRoom should validate tile structure before acceptance."""
    room = DeadbandRoom()
    malformed_tile = {'phase': 'P1'}  # Missing 'action', 'timestamp', etc.
    success, message = room.submit_tile(malformed_tile)
    self.assertFalse(success)
    self.assertIn('missing', message.lower())
    self.assertEqual(len(room.tiles), 0)  # No tile added
```

### 5. Test: Protocol with Dynamic Constraints Update
**Scenario:** Constraints change between P0 and P1 evaluation (simulating real-time environment).
**Edge:** Protocol should re-evaluate P0 when constraints update mid-cycle.
**Expected:** Room detects constraint change and re-runs P0 mapping.
```python
def test_dynamic_constraints_mid_protocol(self):
    """DeadbandRoom should handle constraint updates during execution."""
    room = DeadbandRoom()
    actions = ['move_forward', 'turn_left', 'turn_right']
    # Initial constraints: 'move_forward' is P0
    constraints = {'blocked': ['move_forward']}
    # Simulate constraint change via callback
    def update_constraints():
        constraints['blocked'] = []  'move_forward' becomes allowed
    room.constraint_update_callback = update_constraints
    result = room.apply_protocol(actions, constraints)
    # After update, 'move_forward' should be in safe channels
    self.assertIn('move_forward', result.get('safe_channels', []))
```
