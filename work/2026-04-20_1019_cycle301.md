## 5 Edge-Case Tests

### 1. Test: Malformed Tile Metadata Causing Serialization Failure
**Scenario**: A tile with non‑JSON‑serializable metadata (e.g., a Python `set`) is submitted.  
**Expected**: Room should catch `TypeError`, log error, reject tile, and maintain room state consistency.  
**Rationale**: Ensures robustness against malformed data from buggy agents.

```python
def test_malformed_tile_metadata_rejection(self):
    """DeadbandRoom should reject tiles with non‑serializable metadata."""
    room = DeadbandRoom()
    tile = Tile(
        phase=1,
        content="Valid content",
        metadata={"valid": "data", "invalid": set([1, 2, 3])}  # set is not JSON‑serializable
    )
    with self.assertLogs(level='ERROR') as log:
        result = room.submit_tile(tile)
        self.assertFalse(result.success)
        self.assertIn("metadata serialization", log.output[0])
    self.assertEqual(len(room.accumulated_tiles), 0)  # No tile accepted
```

### 2. Test: Concurrent Tile Submission Race Condition
**Scenario**: Two threads submit tiles simultaneously while the room is exporting an ensign.  
**Expected**: Room should lock appropriately; one tile may be queued, but room state should not corrupt.  
**Rationale**: DeadbandRoom may be used in async/threaded contexts; basic thread safety is required.

```python
def test_concurrent_submission_during_export(self):
    """Room should handle concurrent tile submission during ensign export."""
    room = DeadbandRoom()
    room.start_ensign_export()  # Simulate export in progress
    
    def submit_tile(phase):
        tile = Tile(phase=phase, content=f"Thread {phase}", metadata={})
        return room.submit_tile(tile)
    
    from concurrent.futures import ThreadPoolExecutor
    with ThreadPoolExecutor(max_workers=2) as executor:
        futures = [executor.submit(submit_tile, i) for i in [1, 2]]
        results = [f.result() for f in futures]
    
    # At least one should succeed; room state intact
    self.assertTrue(any(r.success for r in results))
    self.assertIsNotNone(room.ensign_export)  # Export state unchanged
```

### 3. Test: Phase Violation Recovery (Agent Skips to P2)
**Scenario**: An agent submits a P2 tile before any P0 or P1 tiles exist.  
**Expected**: Room should reject the tile, log a protocol violation, and suggest corrective action (submit P0 first).  
**Rationale**: Enforces strict phase priority—core to Deadband Protocol.

```python
def test_phase_skip_violation_recovery(self):
    """Room must reject P2 tiles when no P0/P1 tiles are present."""
    room = DeadbandRoom()
    tile = Tile(phase=2, content="Optimization attempt", metadata={})
    with self.assertLogs(level='WARNING') as log:
        result = room.submit_tile(tile)
        self.assertFalse(result.success)
        self.assertIn("P0 not mapped", log.output[0])
    
    # After submitting a valid P0 tile, P2 should now be accepted
    p0_tile = Tile(phase=0, content="Negative space", metadata={})
    p0_result = room.submit_tile(p0_tile)
    self.assertTrue(p0_result.success)
    
    p2_result = room.submit_tile(tile)  # Same P2 tile
    self.assertTrue(p2_result.success)  # Now acceptable
```

### 4. Test: Tile Content Size Limit Enforcement
**Scenario**: A tile with content exceeding a configured size limit (e.g., 1 MB) is submitted.  
**Expected**: Room should reject oversized tiles, preserving system resources.  
**Rationale**: Prevents memory exhaustion from erroneous or malicious large submissions.

```python
def test_oversized_tile_rejection(self):
    """Room should enforce a configurable tile size limit."""
    room = DeadbandRoom(max_tile_size=1024)  # 1 KB limit
    large_content = "x" * 2048  # 2 KB, exceeding limit
    tile = Tile(phase=1, content=large_content, metadata={})
    
    result = room.submit_tile(tile)
    self.assertFalse(result.success)
    self.assertIn("exceeds size limit", result.message)
    self.assertEqual(len(room.accumulated_tiles), 0)
```

### 5. Test: Ensign Export with Empty Tile Set
**Scenario**: Request ensign export when no tiles have been accumulated.  
**Expected**: Room should return a meaningful error or empty adapter, not crash.  
**Rationale**: Handles edge case where a room is exported prematurely (e.g., by automated tooling).

```python
def test_ensign_export_with_no_tiles(self):
    """Exporting an ensign from an empty room should not crash."""
    room = DeadbandRoom()
    # No tiles submitted
    with self.assertRaises(RoomExportError) as ctx:
        room.export_ensign()
    self.assertIn("no tiles", str(ctx.exception).lower())
    # Alternative: room could return a "null" ensign; adjust test accordingly
```
