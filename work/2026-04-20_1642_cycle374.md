## New Edge-Case Tests
Here are 5 additional tests for `DeadbandRoom`. Each test is a tile that can be submitted to the PLATO room server.

### Test 1: Concurrent Tile Submissions
**Scenario:** Two agents submit tiles to the same room simultaneously during P1 phase.
**Edge:** Race condition in room state update.
**Assertion:** Room processes both tiles, maintains consistent state, does not corrupt channel mapping.

```python
def test_concurrent_p1_submissions(self):
    """DeadbandRoom should handle simultaneous P1 tile submissions without state corruption."""
    room = DeadbandRoom(room_id="concurrent_test")
    room.current_phase = "P1"
    
    # Simulate concurrent submissions
    tile_a = {"phase": "P1", "agent": "agent_a", "channels": ["channel_x"]}
    tile_b = {"phase": "P1", "agent": "agent_b", "channels": ["channel_y"]}
    
    # In real use, would use threading; here we simulate sequential with state checks
    room.submit_tile(tile_a)
    state_after_a = room.get_state()
    room.submit_tile(tile_b)
    state_after_b = room.get_state()
    
    # Both channels should be recorded
    assert "channel_x" in state_after_b["safe_channels"]
    assert "channel_y" in state_after_b["safe_channels"]
    # State should be consistent (no duplicates)
    assert len(state_after_b["safe_channels"]) == 2
```

### Test 2: Protocol Violation – Skip P0
**Scenario:** Agent submits a P1 tile when room is in P0 phase (hasn't mapped negative space).
**Edge:** Room must reject tile and enforce P0-first rule.
**Assertion:** Tile rejected, room remains in P0, error logged.

```python
def test_reject_p1_before_p0(self):
    """DeadbandRoom must reject P1 tiles if P0 negative space not yet mapped."""
    room = DeadbandRoom(room_id="protocol_test")
    room.current_phase = "P0"
    room.negative_space = []  # Empty, not mapped
    
    tile = {"phase": "P1", "agent": "impatient_agent", "channels": ["safe_zone"]}
    
    with pytest.raises(ProtocolViolationError) as excinfo:
        room.submit_tile(tile)
    
    assert "P0 must be completed before P1" in str(excinfo.value)
    assert room.current_phase == "P0"  # Phase unchanged
```

### Test 3: Large Negative Space Map
**Scenario:** Agent submits a P0 tile with a complex, nested negative space map (e.g., 100+ constraints with dependencies).
**Edge:** Room must parse and store large constraint structures without performance issues or data loss.
**Assertion:** All constraints stored, room transitions to P1, structure integrity maintained.

```python
def test_large_p0_negative_space(self):
    """DeadbandRoom should handle large, complex negative space maps."""
    room = DeadbandRoom(room_id="large_p0_test")
    
    # Generate a large negative space map
    large_negative_space = []
    for i in range(150):
        constraint = {
            "id": f"constraint_{i}",
            "type": "forbidden",
            "condition": f"value > {i}",
            "dependencies": [f"constraint_{j}" for j in range(max(0, i-5), i)]
        }
        large_negative_space.append(constraint)
    
    tile = {"phase": "P0", "agent": "mapper_agent", "negative_space": large_negative_space}
    
    room.submit_tile(tile)
    
    assert room.current_phase == "P1"
    assert len(room.negative_space) == 150
    # Check a sample constraint integrity
    assert room.negative_space[75]["id"] == "constraint_75"
    assert len(room.negative_space[75]["dependencies"]) == 5
```

### Test 4: Partial Channel Optimization (P2)
**Scenario:** Safe channels are found in P1, but some are invalid for optimization (e.g., incomplete data). Room enters P2.
**Edge:** Optimization should only operate on valid channels; invalid channels should be logged but ignored.
**Assertion:** P2 optimization runs only on valid channels, results recorded, invalid channels noted.

```python
def test_p2_partial_channels(self):
    """DeadbandRoom P2 optimization should handle partially valid safe channels."""
    room = DeadbandRoom(room_id="partial_p2_test")
    room.current_phase = "P2"
    # Mixed valid and invalid channels
    room.safe_channels = [
        {"id": "channel_a", "metrics": {"throughput": 100}},
        {"id": "channel_b", "metrics": {}},  # Missing required metric
        {"id": "channel_c", "metrics": {"throughput": 50}}
    ]
    
    tile = {"phase": "P2", "agent": "optimizer_agent", "optimization": "greedy"}
    
    room.submit_tile(tile)
    
    # Optimization should run on channel_a and channel_c only
    state = room.get_state()
    assert "channel_a" in state["optimized_paths"]
    assert "channel_c" in state["optimized_paths"]
    assert "channel_b" not in state["optimized_paths"]
    assert "invalid_channels" in state
    assert "channel_b" in state["invalid_channels"]
```

### Test 5: Malformed Tile Validation
**Scenario:** Agent submits a tile with invalid JSON structure, missing required fields, or wrong data types.
**Edge:** Room must reject tile gracefully, not crash, and log validation error.
**Assertion:** Tile rejected with clear error, room state unchanged.

```python
def test_malformed_tile_rejection(self):
    """DeadbandRoom must reject malformed tiles and maintain state integrity."""
    room = DeadbandRoom(room_id="validation_test")
    initial_state = room.get_state()
    
    malformed_tiles = [
        {"phase": "P0"},  # Missing 'agent' and 'negative_space'
        {"agent": "test", "negative_space": []},  # Missing 'phase'
        {"phase": "P1", "agent": 123, "channels": []},  # Wrong type for agent
        "not a dict",  # Not even a dict
    ]
    
    for tile in malformed_tiles:
        with pytest.raises(TileValidationError) as excinfo:
            room.submit_tile(tile)
        assert "invalid tile" in str(excinfo.value).lower()
    
    # State should remain unchanged
    assert room.get_state() == initial_state
```
