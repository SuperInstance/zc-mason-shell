### Test 1: Negative Deadband Threshold
**Scenario**: Deadband threshold provided as negative value.  
**Rationale**: Threshold should be non-negative; negative may indicate configuration error.  
**Expected**: Room should handle gracefully (raise ValueError or clamp to zero).  
```python
def test_negative_deadband_threshold(deadband_room):
    """Deadband threshold should reject negative values."""
    room = deadband_room(deadband_threshold=-5.0)
    with pytest.raises(ValueError):
        room.apply_deadband([1.0, 2.0, 3.0])
```

### Test 3: All Values Outside Deadband (No P1 Channels)
**Scenario**: Every input value is outside deadband threshold relative to reference.  
**Rationale**: Tests P1 failure case—no safe channels found.  
**Expected**: `find_safe_channels()` returns empty list; optimization step skipped.

### Test 4: Single Safe Channel at Boundary
**Scenario**: One value exactly at deadband threshold boundary.  
**Rationale**: Boundary conditions often reveal off-by-one errors.  
**Expected**: Value at threshold may be considered inside or outside based on inclusive/exclusive logic.  
```python
def test_single_channel_at_threshold_boundary(deadband_room):
    """Value exactly at deadband threshold should be consistently handled."""
    room = deadband_room(deadband_threshold=2.0)
    values = [2.0]  # Exactly at threshold from reference 0.0
    channels = room.find_safe_channels(values)
    # Implementation may treat as safe (inclusive) or not (exclusive)
```

### Test 5: Large Input List Performance
**Scenario**: Input list of 10,000 elements.  
**Rationale**: Tests performance and memory handling for realistic data sizes.  
**Expected**: Room should process within reasonable time without memory error.  
```python
def test_large_input_list_performance(deadband_room):
    """Room should handle large input lists efficiently."""
    import time
    room = deadband_room(deadband_threshold=1.0)
    large_list = list(range(10000))  # 0 to 9999
    start = time.time()
    result = room.apply_deadband(large_list)
    elapsed = time.time() - start
    # Should complete under 1 second for 10k elements
    assert elapsed < 1.0
    # Result should be filtered list
    assert len(result) <= len(large_list)
```
