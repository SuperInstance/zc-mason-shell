## Analysis of Existing Test File (`test_deadband_room.py`)
The existing test suite covers:
1. **Basic initialization** – verifies room loads with correct preset name.
2. **P0 negative space mapping** – tests that invalid actions are rejected.
3. **P1 safe channel discovery** – tests that valid actions within a discovered channel are accepted.
4. **P2 optimization within channel** – tests that an optimal action within a known channel is accepted.
5. **Protocol violation** – tests that skipping directly to P2 (without P0/P1) is rejected.

The tests use a mock agent context and assert on the room's response messages and internal state (`current_phase`). The room's logic is stateful: it tracks `current_phase` (P0, P1, P2) and `safe_channels` (a list).

## Identified Edge Cases (P0 – Negative Space)
From the Deadband Protocol: P0 is "what NOT to do". Edge cases are scenarios where the room's behavior at boundaries or under unusual input should be predictable and safe.

### Edge Case 1: Empty Action String
**Scenario:** Agent submits an empty string as action.
**Risk:** The room's action parsing may assume a non‑empty string.
**Test Goal:** Verify the room handles empty input gracefully (should reject as invalid P0).

### Edge Case 2: Maximum Safe Channels Limit
**Scenario:** The `safe_channels` list grows beyond a practical limit (e.g., 1000 channels).
**Risk:** Memory or performance degradation; possible denial‑of‑service.
**Test Goal:** Verify the room either enforces a limit or remains stable under high channel count.

### Edge Case 3: Phase Transition with No Safe Channels
**Scenario:** Agent attempts to move from P0 to P1 when `safe_channels` is empty.
**Risk:** The room might allow P1 without any channels, violating the protocol.
**Test Goal:** Verify P1 is only reachable if at least one safe channel exists.

### Edge Case 4: Malformed Channel Syntax
**Scenario:** Agent submits a channel string with invalid characters or structure.
**Risk:** Parsing error could crash the room or corrupt state.
**Test Goal:** Verify malformed channel strings are rejected in P1.

### Edge Case 5: Concurrent Agent Interactions
**Scenario:** Two agents interact with the same room instance simultaneously (simulated).
**Risk:** Race conditions on `current_phase` or `safe_channels`.
**Test Goal:** Verify thread‑safety or document that the room is not thread‑safe.

import pytest
from plato_torch.rooms.deadband_room import DeadbandRoom

class TestDeadbandRoomEdgeCases:
    """Edge‑case tests for DeadbandRoom."""

    def test_empty_action_string(self):
        """P0: Empty action string should be rejected as invalid."""
        room = DeadbandRoom()
        agent_context = {"agent_id": "test_agent"}
        
        response = room.handle_action("", agent_context)
        
        assert "invalid" in response.lower() or "reject" in response.lower()
        assert room.current_phase == "P0"  # Should remain in P0

    def test_max_safe_channels_limit(self):
        """P1: Room should handle large number of safe channels without crashing."""
        room = DeadbandRoom()
        agent_context = {"agent_id": "test_agent"}
        
        # First, discover many safe channels
        for i in range(100):
            channel_action = f"discover channel_{i}"
            room.handle_action(channel_action, agent_context)
        
        # Verify room still functions
        assert len(room.safe_channels) == 100
        # Attempt a P2 optimization within a known channel
        response = room.handle_action("optimize channel_50", agent_context)
        assert "optimize" in response.lower() or "accepted" in response.lower()
        assert room.current_phase == "P2"

    def test_phase_transition_no_channels(self):
        """P0→P1: Should not transition to P1 if no safe channels exist."""
        room = DeadbandRoom()
        agent_context = {"agent_id": "test_agent"}
        
        # Force P0 (default)
        assert room.current_phase == "P0"
        
        # Attempt to move to P1 without discovering any channel
        response = room.handle_action("enter P1", agent_context)
        
        # Should reject or remain in P0
        assert "cannot" in response.lower() or "invalid" in response.lower() or "reject" in response.lower()
        assert room.current_phase == "P0"

    def test_malformed_channel_syntax(self):
        """P1: Malformed channel strings should be rejected."""
        room = DeadbandRoom()
        agent_context = {"agent_id": "test_agent"}
        
        # First, get to P1 by discovering a valid channel
        room.handle_action("discover valid_channel", agent_context)
        assert room.current_phase == "P1"
        
        # Attempt to add a malformed channel
        malformed = "discover channel!@#$"
        response = room.handle_action(malformed, agent_context)
        
        assert "invalid" in response.lower() or "reject" in response.lower()
        # Existing safe channels should remain unchanged
        assert "valid_channel" in room.safe_channels
        assert "channel!@#$" not in room.safe_channels

    def test_concurrent_agent_simulation(self):
        """Simulate two agents acting sequentially; verify state consistency."""
        room = DeadbandRoom()
        agent1_context = {"agent_id": "agent_1"}
        agent2_context = {"agent_id": "agent_2"}
        
        # Agent 1 discovers a channel
        room.handle_action("discover channel_a", agent1_context)
        assert room.current_phase == "P1"
        assert "channel_a" in room.safe_channels
        
        # Agent 2 attempts to optimize within that channel
        response = room.handle_action("optimize channel_a", agent2_context)
        
        # Room should accept (channel is known) and move to P2
        assert "optimize" in response.lower() or "accepted" in response.lower()
        assert room.current_phase == "P2"
        # Safe channels list should persist across agents
        assert "channel_a" in room.safe_channels
