### Test 1: Empty State Space with Non-Zero Deadband
**Scenario:** What happens when the state space is empty (`[]`) but deadband threshold is positive?
**Edge:** Tests initialization boundary where no valid states exist from the start.
**Expected:** Room should initialize but P1 should return empty channels.

```python
def test_empty_state_space_with_deadband():
    """DeadbandRoom should handle empty initial state space gracefully."""
    room = DeadbandRoom(
        state_space=[],
        deadband_threshold=0.1,
        max_iterations=100
    )
    # Should initialize without error
    assert room.deadband_threshold == 0.1
    # P1 should return empty result
    channels = room.apply_p1([])
    assert channels == []
```

### Test 2: Single-State Space with High Deadband
**Scenario:** Only one possible state exists, deadband threshold exceeds possible variation.
**Edge:** Tests protocol behavior when optimization (P2) has no room to operate.
**Expected:** P0 should pass, P1 should return the single state, P2 should return same state.

```python
def test_single_state_high_deadband():
    """Protocol should handle single-state spaces where deadband prevents movement."""
    room = DeadbandRoom(
        state_space=[{"x": 0.5}],
        deadband_threshold=1.0,  # Larger than possible variation
        max_iterations=50
    )
    
    # P0: No negative space to map (empty list)
    negative_space = room.apply_p0([])
    assert negative_space == []
    
    # P1: Only channel is the single state
    channels = room.apply_p1([])
    assert len(channels) == 1
    assert channels[0] == {"x": 0.5}
    
    # P2: Optimization should return same state (can't move beyond deadband)
    optimized = room.apply_p2(channels[0])
    assert optimized == {"x": 0.5}
```

### Test 3: Protocol Violation with Custom Violation Handler
**Scenario:** Agent skips to P2, but room has custom violation callback.
**Edge:** Tests that custom violation handlers are properly invoked.
**Expected:** Custom handler should be called with violation details.

```python
def test_protocol_violation_custom_handler():
    """Custom violation handlers should be invoked when protocol is violated."""
    violation_log = []
    
    def custom_handler(phase_skipped, current_state):
        violation_log.append({
            "skipped": phase_skipped,
            "state": current_state
        })
    
    room = DeadbandRoom(
        state_space=[{"x": i*0.1} for i in range(10)],
        deadband_threshold=0.05,
        max_iterations=100,
        violation_callback=custom_handler
    )
    
    # Simulate agent incorrectly trying to apply P2 directly
    try:
        room.apply_p2({"x": 0.3})
    except ProtocolViolation:
        pass  # Expected
    
    # Custom handler should have been called
    assert len(violation_log) == 1
    assert violation_log[0]["skipped"] == "P0,P1"
    assert violation_log[0]["state"] == {"x": 0.3}
```

### Test 5: Deadband Threshold of Zero
**Scenario:** Zero deadband means any movement is significant.
**Edge:** Tests boundary condition where deadband protocol becomes strict optimization.
**Expected:** P1 should return all states, P2 should optimize freely.

```python
def test_zero_deadband_threshold():
    """Zero deadband should allow free movement within state space."""
    state_space = [{"x": i*0.2} for i in range(5)]  # [0.0, 0.2, 0.4, 0.6, 0.8]
    room = DeadbandRoom(
        state_space=state_space,
        deadband_threshold=0.0,  # Zero tolerance
        max_iterations=100
    )
    
    # P0: Assume no negative space
    negative_space = room.apply_p0([])
    
    # P1: With zero deadband, all states might be considered channels
    # (or implementation might treat this specially)
    channels = room.apply_p1(negative_space)
    
    # P2: Should be able to move to any state (no deadband restriction)
    current = {"x": 0.4}
    optimized = room.apply_p2(current)
    
    # With zero deadband, optimization might move to any better state
    # This tests that zero threshold doesn't cause division by zero or other errors
    assert optimized != current  # Might change, or might stay if already optimal
    assert any(s["x"] == optimized["x"] for s in state_space)  # Must be valid state
```
