## 5 Additional Edge-Case Tests

Based on the Deadband Protocol doctrine (P0 → P1 → P2, strict priority) and the room's API, I propose these edge cases:

### Test 1: **Empty Negative Space (P0)**
**Scenario**: An agent submits a P0 report that maps *no* negative space (empty list).
**Edge Case**: Is an empty P0 considered valid? The protocol says "map negative space" – does mapping "nothing" violate the intent?
**Test Purpose**: Verify the room handles empty P0 input without crashing and proceeds to P1.
**Expected**: Room accepts empty P0, updates state, transitions to P1.

### Test 2: **Overlapping Safe Channels (P1)**
**Scenario**: An agent submits a P1 report where safe channels overlap or are contradictory.
**Edge Case**: The protocol doesn't specify uniqueness or non-overlap of channels.
**Test Purpose**: Ensure the room can handle overlapping channels (e.g., merge them, keep both, or reject).
**Expected**: Based on the code, it likely stores them as-is; test that state updates correctly.

### Test 3: **P2 Optimization Without P1 Channels**
**Scenario**: An agent attempts to submit a P2 optimization before any P1 safe channels are defined.
**Edge Case**: Strict priority means P2 requires P1. Does the room enforce this mid-session?
**Test Purpose**: Verify the room rejects P2 if current_phase is not P2 (i.e., P1 not completed).
**Expected**: Room raises a ProtocolViolation error.

### Test 4: **Massive Input Data**
**Scenario**: An agent submits a P0 report with an extremely large list of negative space items (e.g., 10,000 entries).
**Edge Case**: Performance and memory handling.
**Test Purpose**: Ensure the room processes large inputs without timeout or memory overflow.
**Expected**: Room processes successfully, though may log a warning.

### Test 5: **Malformed Tile Submission**
**Scenario**: An agent submits a tile that is not a dictionary or lacks required fields (`work_output`, `agent_id`, `timestamp`).
**Edge Case**: The room's `submit_tile` method expects a specific structure.
**Test Purpose**: Verify robust validation and appropriate error messages.
**Expected**: Room raises a ValueError or similar with a descriptive message.

def test_p0_empty_negative_space():
    """Edge case: P0 report with empty negative space list."""
    room = DeadbandRoom()
    agent_id = "test_agent"
    
    # Start session
    room.start_session(agent_id)
    
    # Submit P0 with empty list
    p0_report = {"negative_space": []}
    result = room.submit_phase_report(agent_id, "P0", p0_report)
    
    # Should accept and move to P1
    assert result["status"] == "accepted"
    assert room.current_phase == "P1"
    assert room.negative_space == []  # Should store empty list

def test_p1_overlapping_safe_channels():
    """Edge case: P1 report with overlapping safe channels."""
    room = DeadbandRoom()
    agent_id = "test_agent"
    
    room.start_session(agent_id)
    room.submit_phase_report(agent_id, "P0", {"negative_space": ["rock1", "rock2"]})
    
    # Submit P1 with overlapping channels
    p1_report = {"safe_channels": [{"area": "zone_a"}, {"area": "zone_a"}]}
    result = room.submit_phase_report(agent_id, "P1", p1_report)
    
    assert result["status"] == "accepted"
    assert room.current_phase == "P2"
    # Check that both channels are stored (no deduplication assumed)
    assert len(room.safe_channels) == 2
    assert room.safe_channels[0]["area"] == "zone_a"
    assert room.safe_channels[1]["area"] == "zone_a"

def test_p2_without_p1_channels():
    """Edge case: Attempt P2 optimization before P1 channels defined."""
    room = DeadbandRoom()
    agent_id = "test_agent"
    
    room.start_session(agent_id)
    room.submit_phase_report(agent_id, "P0", {"negative_space": ["rock1"]})
    # Skip P1, try P2 directly
    p2_report = {"optimizations": [{"path": "fast"}]}
    
    # Should raise an error because current_phase is P1, not P2
    with pytest.raises(ProtocolViolation) as exc_info:
        room.submit_phase_report(agent_id, "P2", p2_report)
    
    assert "P2" in str(exc_info.value) and "P1" in str(exc_info.value)

def test_p0_massive_input():
    """Edge case: P0 report with very large negative space list."""
    room = DeadbandRoom()
    agent_id = "test_agent"
    
    room.start_session(agent_id)
    
    # Generate 10,000 items
    large_list = [f"rock_{i}" for i in range(10000)]
    p0_report = {"negative_space": large_list}
    
    import time
    start = time.time()
    result = room.submit_phase_report(agent_id, "P0", p0_report)
    elapsed = time.time() - start
    
    assert result["status"] == "accepted"
    assert room.current_phase == "P1"
    assert len(room.negative_space) == 10000
    # Performance check: should process within 2 seconds
    assert elapsed < 2.0, f"Processing 10k items took {elapsed:.2f}s"

def test_malformed_tile_submission():
    """Edge case: Submit a tile with missing required fields."""
    room = DeadbandRoom()
    agent_id = "test_agent"
    
    room.start_session(agent_id)
    room.submit_phase_report(agent_id, "P0", {"negative_space": ["rock1"]})
    room.submit_phase_report(agent_id, "P1", {"safe_channels": [{"area": "zone1"}]})
    room.submit_phase_report(agent_id, "P2", {"optimizations": [{"path": "best"}]})
    
    # Submit malformed tile (missing timestamp)
    bad_tile = {"work_output": "some work", "agent_id": agent_id}
    
    with pytest.raises(ValueError) as exc_info:
        room.submit_tile(bad_tile)
    
    assert "missing" in str(exc_info.value).lower() or "required" in str(exc_info.value).lower()
