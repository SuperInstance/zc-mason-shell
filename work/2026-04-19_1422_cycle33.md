### Test 1: Protocol Violation with P2 Before P1
**Scenario**: Agent attempts to submit P2 (optimization) tile without establishing P1 (safe channel) first
**Expected**: Room rejects tile with `ProtocolViolation` error
**Rationale**: Tests strict priority enforcement of Deadband Protocol

```rust
#[test]
fn test_p2_before_p1_violation() {
    let mut room = DeadbandRoom::new();
    let agent_id = AgentId::new("test-agent");
    
    // Agent tries to submit P2 tile without P1
    let tile = Tile::new(
        agent_id.clone(),
        TileType::P2Optimization,
        "Trying to optimize before finding safe channel".to_string(),
    );
    
    let result = room.submit_tile(tile);
    assert!(result.is_err());
    assert_eq!(result.unwrap_err(), RoomError::ProtocolViolation);
}
```

### Test 2: Concurrent Agent State Corruption
**Scenario**: Multiple agents submit tiles simultaneously while room state is being modified
**Expected**: Room maintains consistent state without data races or corruption
**Rationale**: Tests thread safety in concurrent access scenarios

```rust
#[test]
fn test_concurrent_agent_state_integrity() {
    let room = Arc::new(Mutex::new(DeadbandRoom::new()));
    let mut handles = vec![];
    
    for i in 0..10 {
        let room_clone = Arc::clone(&room);
        handles.push(thread::spawn(move || {
            let agent_id = AgentId::new(format!("agent-{}", i));
            let tile = Tile::new(
                agent_id,
                TileType::P0NegativeSpace,
                format!("P0 tile from agent {}", i),
            );
            
            let mut room = room_clone.lock().unwrap();
            room.submit_tile(tile).unwrap();
        }));
    }
    
    for handle in handles {
        handle.join().unwrap();
    }
    
    let room = room.lock().unwrap();
    assert_eq!(room.get_agent_count(), 10);
    // Verify no duplicate agent IDs or corrupted state
}
```

### Test 4: Malformed Tile Content Injection
**Scenario**: Agent submits tile with specially crafted content attempting to exploit parsing
**Expected**: Room sanitizes or rejects malicious content while preserving valid tiles
**Rationale**: Tests input validation and security boundaries

```rust
#[test]
fn test_malformed_content_handling() {
    let mut room = DeadbandRoom::new();
    let agent_id = AgentId::new("malicious-agent");
    
    let test_cases = vec![
        // Null bytes
        "Normal content\0with null byte",
        // Extremely long content
        &"a".repeat(100_000),
        // SQL injection attempt
        "'); DROP TABLE tiles; --",
        // HTML/JS injection
        "<script>alert('xss')</script>",
        // Invalid UTF-8 sequences
        unsafe { std::str::from_utf8_unchecked(&[0xFF, 0xFE]) },
    ];
    
    for (i, content) in test_cases.iter().enumerate() {
        let tile = Tile::new(
            agent_id.clone(),
            TileType::P0NegativeSpace,
            content.to_string(),
        );
        
        let result = room.submit_tile(tile);
        // Room should either accept with sanitization or reject
        // but never crash or corrupt state
        if result.is_err() {
            assert!(matches!(
                result.unwrap_err(),
                RoomError::InvalidTileContent | RoomError::TileTooLarge
            ));
        } else {
            // If accepted, verify content was sanitized
            let tiles = room.get_tiles_by_agent(&agent_id);
            let last_tile = tiles.last().unwrap();
            assert!(!last_tile.content.contains('\0'));
            assert!(last_tile.content.len() <= MAX_TILE_LENGTH);
        }
    }
}
```

### Test 5: Protocol State Machine Edge Transitions
**Scenario**: Agent rapidly transitions between P0, P1, P2 states with minimal tiles
**Expected**: Room correctly tracks protocol state and enforces valid transitions
**Rationale**: Tests state machine robustness under edge conditions

```rust
#[test]
fn test_rapid_protocol_transitions() {
    let mut room = DeadbandRoom::new();
    let agent_id = AgentId::new("transition-agent");
    
    // Valid sequence: P0 -> P1 -> P2
    let tiles = vec![
        (TileType::P0NegativeSpace, "Found rock"),
        (TileType::P1SafeChannel, "Found channel"),
        (TileType::P2Optimization, "Optimized path"),
    ];
    
    for (tile_type, content) in tiles {
        let tile = Tile::new(agent_id.clone(), tile_type, content.to_string());
        room.submit_tile(tile).unwrap();
    }
    
    // Verify final state
    let agent_state = room.get_agent_state(&agent_id).unwrap();
    assert_eq!(agent_state.current_phase, Some(Phase::P2));
    
    // Invalid: Try to go back to P1 after P2
    let back_tile = Tile::new(
        agent_id.clone(),
        TileType::P1SafeChannel,
        "Trying to go back".to_string(),
    );
    let result = room.submit_tile(back_tile);
    assert!(result.is_err());
    assert_eq!(result.unwrap_err(), RoomError::ProtocolViolation);
    
    // Edge: Empty P0 tile (just mapping "no rocks")
    let empty_p0 = Tile::new(
        agent_id.clone(),
        TileType::P0NegativeSpace,
        "".to_string(), // Empty but valid
    );
    assert!(room.submit_tile(empty_p0).is_ok());
}
```
