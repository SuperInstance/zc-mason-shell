## Analysis of Existing DeadbandRoom Implementation

Based on reading `plato-torch/rooms/deadband.py`:

**Core Purpose:** Implements the Deadband Protocol (P0/P1/P2) as a training room for agents to learn constraint navigation.

**Key Components:**
1. `DeadbandRoom` class inherits from `BaseRoom`
2. Three-phase state machine: `P0_MAPPING`, `P1_CHANNEL_FINDING`, `P2_OPTIMIZATION`
3. `ConstraintMap` tracks negative space (P0 obstacles)
4. `SafeChannel` objects represent valid navigation paths
5. `step()` method processes agent actions and updates room state

## Identified Test Gaps (Edge Cases)

### Gap 1: Invalid State Transitions
Current tests validate correct transitions but don't test illegal jumps (e.g., P0→P2 skipping P1).

### Gap 2: Constraint Map Edge Cases
Missing tests for overlapping constraints, zero-size constraints, and constraint removal.

### Gap 3: Channel Boundary Conditions
No tests for channels that touch constraint boundaries or channels with zero width.

### Gap 4: Optimization Degradation
Missing tests for what happens when optimization (P2) degrades below P1 safety thresholds.

### Gap 5: Multi-Agent Interaction
Current tests are single-agent; missing tests for multiple agents in same room with conflicting constraints.

## 5 New Edge-Case Tests

```python
# test_deadband_edge_cases.py
import pytest
import numpy as np
from plato_torch.rooms.deadband import DeadbandRoom, P0_MAPPING, P1_CHANNEL_FINDING, P2_OPTIMIZATION

class TestDeadbandRoomEdgeCases:
    
    def test_illegal_state_transition_throws(self):
        """P0: Test that skipping P1 (going directly P0→P2) raises appropriate error."""
        room = DeadbandRoom(room_id="edge_test_1")
        room.state = P0_MAPPING
        room.constraint_map.add_constraint((0, 0), (10, 10))
        
        # Attempt illegal transition
        with pytest.raises(ValueError, match="Cannot transition from P0_MAPPING to P2_OPTIMIZATION"):
            room.state = P2_OPTIMIZATION
    
    def test_overlapping_constraints_merge(self):
        """P1: Test that overlapping constraints merge into single negative space region."""
        room = DeadbandRoom(room_id="edge_test_2")
        room.state = P0_MAPPING
        
        # Add overlapping constraints
        room.constraint_map.add_constraint((0, 0), (5, 5))
        room.constraint_map.add_constraint((3, 3), (8, 8))
        
        # Should merge to single constraint covering union
        constraints = room.constraint_map.get_all_constraints()
        assert len(constraints) == 1
        assert constraints[0].bounds == ((0, 0), (8, 8))
    
    def test_zero_width_channel_rejection(self):
        """P1: Test that channels with zero safe width are rejected during P1."""
        room = DeadbandRoom(room_id="edge_test_3")
        room.state = P1_CHANNEL_FINDING
        
        # Create a constraint map that leaves no safe space
        room.constraint_map.add_constraint((0, 0), (100, 50))  # Bottom half
        room.constraint_map.add_constraint((0, 50), (100, 100))  # Top half
        
        # Attempt to find channels
        channels = room.find_safe_channels()
        assert len(channels) == 0  # No valid channels should exist
    
    def test_optimization_degradation_fallback(self):
        """P2: Test that when optimization score falls below P1 threshold, agent falls back to P1."""
        room = DeadbandRoom(room_id="edge_test_4")
        room.state = P2_OPTIMIZATION
        
        # Simulate optimization degradation
        initial_score = room.optimization_score
        room.optimization_score = room.P1_SAFETY_THRESHOLD - 0.1  # Below threshold
        
        # Room should detect degradation and trigger fallback
        room._check_optimization_safety()
        assert room.state == P1_CHANNEL_FINDING
    
    def test_multi_agent_constraint_collision(self):
        """P0/P1: Test multiple agents adding constraints that conflict."""
        room = DeadbandRoom(room_id="edge_test_5")
        room.state = P0_MAPPING
        
        # Agent A maps a constraint
        room.process_agent_action(agent_id="agent_a", action={"type": "add_constraint", "bounds": ((0, 0), (30, 30))})
        
        # Agent B tries to map overlapping but non-identical constraint
        room.process_agent_action(agent_id="agent_b", action={"type": "add_constraint", "bounds": ((20, 20), (50, 50))})
        
        # Constraint map should contain merged region
        constraints = room.constraint_map.get_all_constraints()
        assert len(constraints) == 1
        # Merged region should cover both: (0,0) to (50,50)
        assert constraints[0].bounds == ((0, 0), (50, 50))
```

## Test Rationale

1. **Illegal State Transition**: Ensures strict adherence to Deadband Protocol priority (P0→P1→P2).
2. **Overlapping Constraints**: Tests constraint merging logic, which is critical for accurate negative space mapping.
3. **Zero Width Channels**: Validates that P1 correctly identifies when NO safe channels exist.
4. **Optimization Degradation**: Tests the safety fallback mechanism when P2 optimization fails.
5. **Multi-Agent Collision**: Tests constraint merging with multiple agents, simulating real fleet scenarios.
