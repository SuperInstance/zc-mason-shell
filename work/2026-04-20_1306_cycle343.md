## Edge-Case Tests Written

### 1. Test: Agent Submits Tile with Future Timestamp
**Scenario:** An agent submits a tile with a `created_at` timestamp far in the future.
**Expected:** Tile rejected (invalid timestamp).
**Rationale:** Prevents training on synthetic future data.

```python
def test_submit_tile_future_timestamp(test_room, test_agent, tile_fixture):
    """Tile with future timestamp should be rejected."""
    import datetime
    future_time = datetime.datetime.now() + datetime.timedelta(days=1)
    tile = tile_fixture.copy()
    tile["created_at"] = future_time.isoformat()
    
    result = test_room.submit_tile(test_agent.agent_id, tile)
    assert not result["accepted"]
    assert "timestamp" in result["reason"].lower()
```

### 4. Test: Concurrent Tile Submission from Same Agent
**Scenario:** Same agent submits two tiles simultaneously (thread simulation).
**Expected:** One succeeds, second may be queued or rejected based on locking.
**Rationale:** Tests thread safety of room submission logic.

```python
def test_concurrent_submission_same_agent(test_room, test_agent, tile_fixture):
    """Handle concurrent submissions from same agent."""
    import threading
    
    results = []
    def submit():
        result = test_room.submit_tile(test_agent.agent_id, tile_fixture.copy())
        results.append(result)
    
    threads = [threading.Thread(target=submit) for _ in range(2)]
    for t in threads:
        t.start()
    for t in threads:
        t.join()
    
    # At least one should succeed
    accepted_count = sum(1 for r in results if r["accepted"])
    assert accepted_count >= 1
    # Both shouldn't necessarily succeed due to potential locking
```

### 5. Test: Tile with Circular References in Metadata
**Scenario:** Tile metadata contains self-referential dictionary.
**Expected:** Tile rejected (non-serializable).
**Rationale:** Prevents serialization crashes during tile storage.

```python
def test_submit_tile_circular_reference(test_room, test_agent, tile_fixture):
    """Tile with circular reference in metadata should be rejected."""
    circular_dict = {}
    circular_dict["self"] = circular_dict  # Create circular reference
    tile = tile_fixture.copy()
    tile["metadata"] = circular_dict

    # Should fail during validation/serialization
    assert "serializ" in result["reason"].lower() or "valid" in result["reason"].lower()
```
