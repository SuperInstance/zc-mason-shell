## New Edge-Case Tests

### 1. Test: P0 to P2 Direct Transition (Protocol Violation)
**Purpose:** Verify the room correctly blocks skipping from P0 (negative space mapping) directly to P2 (optimization) without passing through P1 (safe channels). This enforces strict priority.
```python
def test_p0_to_p2_direct_transition_blocked():
    """Attempt to transition from P0 directly to P2 should raise InvalidTransitionError."""
    room = DeadbandRoom()
    room.state = "P0"
    with pytest.raises(InvalidTransitionError, match="Cannot skip P1"):
        room.transition("P2")
    assert room.state == "P0"  # State unchanged
```

### 2. Test: Empty Action Queue with Non-Empty State
**Purpose:** Test behavior when `process_actions()` is called with an empty action list while in P1 or P2 states. Should maintain state but log a warning.
```python
def test_empty_action_queue_in_p1():
    """Processing empty actions in P1 should log warning but not change state."""
    room = DeadbandRoom()
    room.state = "P1"
    with caplog.at_level(logging.WARNING):
        result = room.process_actions([])

    assert room.state == "P1"
```

### 5. Test: Concurrent State Transition Race Condition
**Purpose:** Simulate two threads attempting to transition state simultaneously. Ensure thread safety via locks.
```python
def test_concurrent_state_transition():
    """Multiple threads attempting transition should serialize safely."""
    room = DeadbandRoom()
    room.state = "P0"
    errors = []
    
    def attempt_transition(target):
        try:
            room.transition(target)
        except Exception as e:
            errors.append(str(e))
    
    threads = [
        threading.Thread(target=attempt_transition, args=("P1",)),
        threading.Thread(target=attempt_transition, args=("P1",))
    ]
    for t in threads:
        t.start()
    for t in threads:
        t.join()
    
    # Only one should succeed, other might get InvalidTransitionError or succeed idempotently
    assert room.state == "P1"
    # Verify no crashes or undefined behavior
    assert len(errors) <= 1
```
