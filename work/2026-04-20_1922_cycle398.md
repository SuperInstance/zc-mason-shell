### Test 1: Position Wrapping at Map Boundaries
**Scenario:** Agent moves beyond defined map boundaries.
**Rationale:** The environment uses a toroidal (wrapping) map. Need to verify position wraps correctly at both positive and negative extremes.
```python
def test_position_wrapping_extremes():
    """Test that position wraps correctly at map boundaries."""
    room = DeadbandRoom(map_size=10)
    # Force position beyond positive boundary
    room.state = {"position": 12, "velocity": 0, "channel": 0, "phase": 0}
    room._apply_wrapping()
    assert room.state["position"] == 2
    # Force position beyond negative boundary
    room.state = {"position": -5, "velocity": 0, "channel": 0, "phase": 0}
    room._apply_wrapping()
    assert room.state["position"] == 5
```

### Test 2: Velocity Saturation with Repeated Actions
**Scenario:** Agent applies boost action multiple times consecutively.
**Rationale:** Velocity has maximum/minimum limits. Need to ensure saturation works and doesn't overflow.
```python
def test_velocity_saturation_boost_spam():
    """Test velocity saturation when boost action is spammed."""
    room = DeadbandRoom(max_velocity=5)
    room.state = {"position": 0, "velocity": 0, "channel": 0, "phase": 0}
    # Apply boost 10 times (exceeds max velocity)
    for _ in range(10):
        room._apply_action(2)  # boost action
    assert room.state["velocity"] == 5  # Should saturate at max
    # Test negative saturation
    room.state["velocity"] = 0
    for _ in range(10):
        room._apply_action(0)  # left action (negative velocity)
    assert room.state["velocity"] == -5
```

### Test 3: Phase Transition with Invalid Channel
**Scenario:** Agent attempts P2 optimization while in an invalid channel.
**Rationale:** According to Deadband Protocol, P2 should only execute when in a valid channel (P1 satisfied). Need to verify phase logic enforces this.
```python
def test_phase_transition_invalid_channel():
    """Test phase transition blocked when channel is invalid."""
    room = DeadbandRoom()
    # Set to invalid channel (channel = -1 indicates no safe channel)
    room.state = {"position": 0, "velocity": 0, "channel": -1, "phase": 1}
    # Attempt P2 optimization action
    reward, done = room.step(2)
    # Should remain in P1 (finding channels) not transition to P2
    assert room.state["phase"] == 1
    # Should receive negative reward for protocol violation
    assert reward < 0
```
