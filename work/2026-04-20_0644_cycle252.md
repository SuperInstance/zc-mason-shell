## 5 Additional Edge-Case Tests

### Test 1: Empty Negative Space (P0 with zero constraints)
**Scenario**: Agent attempts P0 mapping with an empty constraint set
**Edge Case**: What happens when there are no "what NOT to do" rules?
**Expected Behavior**: Room should handle gracefully, possibly allowing all channels in P1
**Test Purpose**: Validate protocol robustness with minimal constraints

```python
def test_p0_empty_negative_space(self):
    """Test DeadbandRoom handles empty P0 constraint set gracefully."""
    room = DeadbandRoom()
    agent_state = {"phase": "P0", "constraints": []}
    
    result = room.process_phase(agent_state)
    
    # Should not crash with empty constraints
    assert result is not None
    assert "status" in result
    # Empty P0 might allow all channels in P1
    assert result.get("allowed_channels") == "all" or result.get("status") == "proceed_to_p1"
```

### Test 2: Overlapping Constraint Channels (P1 conflict resolution)
**Scenario**: P0 constraints create overlapping prohibited zones in P1
**Edge Case**: When safe channels have partial overlaps with constraints
**Expected Behavior**: Room should resolve conflicts using constraint priority or intersection logic
**Test Purpose**: Test channel conflict resolution algorithm

```python
def test_p1_overlapping_constraint_channels(self):
    """Test DeadbandRoom resolves overlapping constraint channels in P1."""
    room = DeadbandRoom()
    agent_state = {
        "phase": "P1",
        "p0_constraints": [
            {"type": "avoid", "zone": "A-B"},
            {"type": "avoid", "zone": "B-C"}
        ],
        "available_channels": ["A", "B", "C", "D"]
    }
    
    result = room.find_safe_channels(agent_state)
    
    # Should resolve overlapping constraints
    assert "safe_channels" in result
    safe_channels = result["safe_channels"]
    # Channel B appears in both constraint zones A-B and B-C
    # Resolution logic should determine if B is safe or not
    assert isinstance(safe_channels, list)
    # At minimum, channel D should be safe (no constraints)
    assert "D" in safe_channels
```

### Test 3: P2 Optimization with Single-Channel Constraint
**Scenario**: P1 yields only one safe channel
**Edge Case**: No optimization choices available in P2
**Expected Behavior**: P2 should still execute but return the single channel as "optimal"
**Test Purpose**: Test P2 behavior with no optimization choices

```python
def test_p2_single_channel_optimization(self):
    """Test P2 optimization when only one safe channel exists."""
    room = DeadbandRoom()
    agent_state = {
        "phase": "P2",
        "p1_channels": ["C3"],  # Only one safe channel
        "optimization_metrics": ["latency", "throughput"]
    }
    
    result = room.optimize_within_channels(agent_state)
    
    # Should handle single-channel case gracefully
    assert "optimal_path" in result
    assert result["optimal_path"] == ["C3"]
    assert "optimization_applied" in result
    # With only one channel, optimization is trivial but should be noted
    assert result["optimization_applied"] == "trivial_single_channel"
```

### Test 4: Protocol Violation with P2 Direct Call (Bypassing P0/P1)
**Scenario**: Agent attempts to call P2 optimization without completing P0 and P1
**Edge Case**: Direct P2 call with missing prerequisite state
**Expected Behavior**: Room should detect protocol violation and return appropriate error
**Test Purpose**: Test strict protocol enforcement beyond simple phase skipping

```python
def test_protocol_violation_missing_prerequisites(self):
    """Test DeadbandRoom detects P2 call without P0/P1 completion."""
    room = DeadbandRoom()
    # Agent tries to optimize without providing P0 constraints or P1 channels
    agent_state = {
        "phase": "P2",
        "optimization_metrics": ["speed"],
        # Missing: p0_constraints, p1_channels
    }
    
    result = room.process_phase(agent_state)
    
    # Should detect protocol violation
    assert "error" in result or "violation" in result
    assert "P0" in str(result.get("error", "")) or "prerequisite" in str(result.get("error", ""))
    # Should not return any optimization results
    assert "optimal_path" not in result
```

### Test 5: Tile Submission with Malformed Protocol Data
**Scenario**: Agent submits a training tile with invalid protocol state data
**Edge Case**: Tile data contradicts protocol rules (e.g., P2 results without P1 channels)
**Expected Behavior**: Tile validation should fail with specific error about protocol consistency
**Test Purpose**: Test tile validation against protocol rules

```python
def test_tile_validation_protocol_inconsistency(self):
    """Test tile validation catches protocol state inconsistencies."""
    room = DeadbandRoom()
    
    # Malformed tile: Claims to be P2 results but has no P1 channels
    malformed_tile = {
        "tile_type": "protocol_experience",
        "phase": "P2",
        "results": {"optimal_path": ["X1", "X2"]},
        "timestamp": "2026-04-20T06:45:00Z",
        # Missing: p1_channels, p0_constraints
    }
    
    validation_result = room.validate_tile(malformed_tile)
    
    # Should fail validation due to protocol inconsistency
    assert validation_result["valid"] is False
    assert "protocol" in validation_result.get("error", "").lower()
    assert "inconsistent" in validation_result.get("error", "").lower() or \
           "prerequisite" in validation_result.get("error", "").lower()
```
