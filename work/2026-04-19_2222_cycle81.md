### Test 1: P0 Violation — Skipping Directly to P2
**Scenario:** Agent attempts optimization (P2) without mapping negative space (P0) or finding safe channels (P1).
**Expected:** Room should reject the action, log a violation, and possibly reset agent state.
```python
def test_p0_violation_skip_to_p2():

    agent = room.create_agent()

    assert "P0 not satisfied" in result["message"]
```

### Test 3: Concurrent Protocol Phases Across Multiple Agents
**Scenario:** Multiple agents in the same room at different protocol phases.

```python
def test_concurrent_phases_multi_agent():
```
