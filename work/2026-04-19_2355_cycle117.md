## 5 Additional Edge-Case Tests

### 1. Test: Malformed JSON Tile
**Scenario:** Tile payload is not valid JSON (e.g., truncated, syntax error).  
**Expected:** Room should reject tile gracefully, log error, not crash.  
**Rationale:** Network or serialization errors can produce invalid JSON.

```python
def test_process_tile_malformed_json(self):
    """DeadbandRoom should handle malformed JSON without crashing."""
    room = DeadbandRoom()
    malformed_tile = "{invalid json, missing quotes}"
    result = room.process_tile(malformed_tile)
    self.assertFalse(result["accepted"])
    self.assertIn("error", result)
```

### 2. Test: Empty Content Tile
**Scenario:** Tile has valid JSON but `content` field is empty string or only whitespace.  
**Expected:** Tile should be rejected (no meaningful training signal).  
**Rationale:** Ensures tiles provide substantive content for training.

```python
def test_process_tile_empty_content(self):
    """Tiles with empty or whitespace-only content should be rejected."""
    room = DeadbandRoom()
    empty_content_tile = {
        "agent_id": "test_agent",
        "timestamp": 1713567890,
        "priority": "P1",
        "content": "   "
    }
    result = room.process_tile(empty_content_tile)
    self.assertFalse(result["accepted"])
    self.assertEqual(result.get("reason"), "empty_content")
```

### 3. Test: Extreme Timestamp Values
**Scenario:** Tile timestamp is zero, negative, or far future (year 3000).  
**Expected:** Timestamp validation should reject unreasonable values.  
**Rationale:** Prevents corrupted or malicious timestamp data from affecting room state.

```python
def test_process_tile_extreme_timestamp(self):
    """Tiles with unrealistic timestamps should be rejected."""
    room = DeadbandRoom()
    future_tile = {
        "agent_id": "test_agent",
        "timestamp": 32503680000,  # year 3000
        "priority": "P0",
        "content": "Avoid null pointer derefs."
    }
    result = room.process_tile(future_tile)
    self.assertFalse(result["accepted"])
    # Room may accept only timestamps within reasonable delta from current time
```

### 4. Test: Priority Boundary – P0 Tile with P2-like Content
**Scenario:** Tile marked P0 (negative space) but contains content that reads like a P2 optimization (e.g., "Optimize loop by unrolling").  
**Expected:** Tile should be accepted as P0 (priority field dictates category).  
**Rationale:** Priority is explicit; content semantic analysis is secondary.

```python
def test_process_tile_priority_boundary(self):
    """Priority field overrides content semantics."""
    room = DeadbandRoom()
    boundary_tile = {
        "agent_id": "test_agent",
        "timestamp": 1713567890,
        "priority": "P0",
        "content": "Optimize matrix multiplication with CUDA kernels."
    }
    result = room.process_tile(boundary_tile)
    self.assertTrue(result["accepted"])
    # Verify it's stored in P0 state, not P2
    state = room.get_state()
    self.assertIn("Optimize matrix multiplication", state["P0"])
```

### 5. Test: Batch Processing State Persistence
**Scenario:** Send multiple valid tiles across separate `process_tile` calls, then check aggregated state.  
**Expected:** State should accumulate across calls, not reset.  
**Rationale:** Ensures room maintains memory across training cycles.

```python
def test_state_persistence_across_batches(self):
    """Room state should accumulate tiles across multiple processing calls."""
    room = DeadbandRoom()
    tiles = [
        {"agent_id": "a1", "timestamp": 1713567890, "priority": "P0", "content": "Don't use raw pointers."},
        {"agent_id": "a2", "timestamp": 1713567891, "priority": "P1", "content": "Use RAII for safety."},
        {"agent_id": "a3", "timestamp": 1713567892, "priority": "P2", "content": "Prefetch data for speed."}
    ]
    for tile in tiles:
        room.process_tile(tile)
    
    state = room.get_state()
    self.assertIn("Don't use raw pointers", state["P0"])
    self.assertIn("Use RAII for safety", state["P1"])
    self.assertIn("Prefetch data for speed", state["P2"])
```
