## 5 Edge-Case Tests for DeadbandRoom

### Test 1: Protocol Skipping Enforcement
**Scenario:** Agent attempts to jump directly to P2 optimization without completing P0 and P1.
**Edge Case:** Verifies strict priority enforcement even when P2 action is technically valid.
**Expected:** Room rejects P2 action and returns to P1 channel identification phase.

```python
def test_protocol_skip_rejection(self):
    """Test that skipping P0→P1→P2 sequence is rejected."""
    room = DeadbandRoom()
    # Agent tries to submit P2 optimization immediately
    result = room.process_action({"phase": "P2", "action": "optimize", "params": {"path": "fastest"}})
    assert result["status"] == "rejected"
    assert "P0" in result["message"]  # Should indicate need to start at P0
    assert room.current_phase == "P0"  # Room resets to initial phase
```

### Test 2: Empty Negative Space Mapping
**Scenario:** Agent submits empty P0 map (no constraints identified).
**Edge Case:** Validates handling of degenerate constraint sets.
**Expected:** Room should either accept empty map as valid (no constraints) or provide guidance, not crash.

```python
def test_empty_p0_map(self):
    """Test handling of empty negative space declaration."""
    room = DeadbandRoom()
    # Complete P0 with empty constraints
    result = room.process_action({"phase": "P0", "action": "map", "constraints": []})
    assert result["status"] == "accepted"
    assert "channels" in result  # Should proceed to P1 with available channels
    # Verify P1 phase is reachable
    result = room.process_action({"phase": "P1", "action": "explore", "channel": "default"})
    assert result["status"] == "accepted"
```

### Test 3: Concurrent Channel Collision
**Scenario:** Multiple agents attempt to claim the same P1 channel simultaneously.
**Edge Case:** Tests room's handling of resource contention.
**Expected:** Room should manage channel allocation (first-come, first-served or with conflict resolution).

```python
def test_concurrent_channel_collision(self):
    """Test channel allocation with simulated concurrent access."""
    room = DeadbandRoom()
    # Agent A claims channel X
    room.process_action({"phase": "P1", "action": "claim", "channel": "channel_x"})
    # Agent B attempts to claim same channel
    result = room.process_action({"phase": "P1", "action": "claim", "channel": "channel_x"})
    assert result["status"] == "rejected"
    assert "occupied" in result["message"].lower()
    # Agent B should be offered alternative channels
    assert "available_channels" in result
```

### Test 4: P2 Optimization with Invalid Bounds
**Scenario:** Agent provides P2 optimization parameters outside P1 channel boundaries.
**Edge Case:** Tests that optimization respects channel constraints.
**Expected:** Room should reject optimization that violates channel bounds.

```python
def test_p2_out_of_bounds(self):
    """Test P2 optimization that exceeds P1 channel boundaries."""
    room = DeadbandRoom()
    # Setup: Complete P0 and P1
    room.process_action({"phase": "P0", "action": "map", "constraints": ["rock"]})
    room.process_action({"phase": "P1", "action": "claim", "channel": "safe_path"})
    # Attempt P2 with parameter outside channel
    result = room.process_action({
        "phase": "P2", 
        "action": "optimize", 
        "params": {"speed": 150, "bounds": {"min": 0, "max": 100}}
    })
    assert result["status"] == "rejected"
    assert "bounds" in result["message"].lower()
    assert room.current_phase == "P1"  # Should revert to channel phase
```

### Test 5: State Recovery After Malformed Action
**Scenario:** Agent sends malformed JSON or missing required fields.
**Edge Case:** Tests room's resilience to protocol violations.
**Expected:** Room should handle parse errors gracefully without corrupting state.

```python
def test_malformed_action_recovery(self):
    """Test recovery after receiving malformed action data."""
    room = DeadbandRoom()
    initial_state = room.get_state()
    
    # Send malformed action (missing phase field)
    try:
        result = room.process_action({"action": "map"})  # Missing "phase"
    except (KeyError, ValueError) as e:
        # Room should catch and handle gracefully
        result = {"status": "error", "message": str(e)}
    
    assert result["status"] == "error"
    # Verify room state is preserved
    recovered_state = room.get_state()
    assert recovered_state == initial_state
    # Room should still be operational
    valid_result = room.process_action({"phase": "P0", "action": "map", "constraints": ["test"]})
    assert valid_result["status"] == "accepted"
```
