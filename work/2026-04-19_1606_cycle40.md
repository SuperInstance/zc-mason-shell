## 5 Additional Edge-Case Tests

Based on the Deadband Protocol doctrine (P0→P1→P2 strict priority) and the existing implementation, here are 5 edge-case tests:

### Test 1: **P0 Violation with Multiple Actions**
**Scenario:** Agent submits multiple actions where some violate P0 constraints.
**Expected:** Room should reject ALL actions (not just the violating ones) and return P0 feedback.
**Rationale:** Tests that P0 violations block entire action batches, not partial execution.

```python
def test_deadband_room_p0_violation_multiple_actions(self):
    """Test that P0 violations reject entire action batch."""
    room = DeadbandRoom()
    state = room.reset()
    
    # Mix of valid and P0-violating actions
    actions = [
        {"type": "move", "direction": "north", "confidence": 0.8},
        {"type": "move", "direction": "into_wall", "confidence": 0.9},  # P0 violation
        {"type": "observe", "target": "door", "confidence": 0.7}
    ]
    
    next_state, reward, done, info = room.step(actions)
    
    # Should reject all actions due to P0 violation
    assert reward < 0  # Negative reward for P0 violation
    assert "p0_violation" in info
    assert info["p0_violation"] == "into_wall"
    assert state["position"] == next_state["position"]  # No state change
```

### Test 2: **P1 Channel Exhaustion**
**Scenario:** All available P1 channels lead to dead ends.
**Expected:** Room should return P1 feedback indicating no safe channels, preventing P2 optimization.
**Rationale:** Tests the protocol's behavior when P1 yields empty results.

```python
def test_deadband_room_p1_no_safe_channels(self):
    """Test behavior when no P1 safe channels exist."""
    room = DeadbandRoom()
    state = room.reset()
    
    # Set up state where all directions are blocked
    room._state["blocked_directions"] = ["north", "south", "east", "west"]
    
    actions = [{"type": "move", "direction": "north", "confidence": 0.9}]
    next_state, reward, done, info = room.step(actions)
    
    assert "p1_no_channels" in info
    assert reward < 0  # Penalty for attempting impossible move
    assert not done  # Should not terminate episode
```

### Test 3: **P2 Optimization with Conflicting Metrics**
**Scenario:** Multiple P2 optimization paths with conflicting reward signals.
**Expected:** Room should apply deterministic tie-breaking (e.g., first valid, highest confidence).
**Rationale:** Tests that P2 doesn't get stuck when metrics conflict.

```python
def test_deadband_room_p2_conflicting_metrics(self):
    """Test P2 optimization with conflicting reward signals."""
    room = DeadbandRoom()
    state = room.reset()
    
    # Two actions with same type but different confidence and reward potential
    actions = [
        {"type": "collect", "resource": "A", "confidence": 0.6, "potential_reward": 10},
        {"type": "collect", "resource": "B", "confidence": 0.9, "potential_reward": 5}
    ]
    
    next_state, reward, done, info = room.step(actions)
    
    # Should select based on deterministic rule (highest confidence first)
    assert "p2_selected" in info
    assert info["p2_selected"] == 1  # Index of selected action
    assert reward == 5  # Reward from selected action B
```

### Test 5: **Long-Running State Corruption**
**Scenario:** Multiple step cycles with malformed actions interspersed.
**Expected:** Room should maintain consistent state despite intermittent bad actions.
**Rationale:** Tests resilience against state corruption over time.

```python
def test_deadband_room_state_consistency_long_run(self):
    """Test state consistency across multiple steps with edge cases."""
    room = DeadbandRoom()
    state = room.reset()
    
    original_position = state["position"]
    
    # Sequence: valid → malformed → valid → P0 violation → valid
    action_sequence = [
        [{"type": "move", "direction": "north", "confidence": 0.8}],
        [{"type": "invalid_format"}],  # Malformed
        [{"type": "move", "direction": "east", "confidence": 0.7}],
        [{"type": "move", "direction": "into_wall", "confidence": 0.9}],  # P0
        [{"type": "move", "direction": "south", "confidence": 0.8}]
    ]
    
    final_position = original_position
    
    for i, actions in enumerate(action_sequence):
        next_state, reward, done, info = room.step(actions)
        
        # Only update position if action was valid and not P0 violation
        if "p0_violation" not in info and "malformed_action" not in info:
            final_position = next_state["position"]
    
    # Final position should reflect only successful moves
    assert room._state["position"] == final_position
    assert room._state["step_count"] == 5  # All steps counted
    assert not done  # Should still be running
```
