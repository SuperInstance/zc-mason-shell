### 2. **Test: Malformed Tile with Nested Circular References**
**Scenario:** Agent submits a tile with self-referential data structures.
**Edge:** JSON serialization/validation might hang or crash.
**Expected:** Graceful rejection with validation error.

```python
def test_malformed_tile_circular_reference():
```

### 4. **Test: Tile Submission with Maximum Size Boundary**
**Scenario:** Agent submits tile at exact size limit.
**Edge:** Off-by-one errors in size validation.
**Expected:** Tile at exact limit should be accepted.

```python
def test_tile_at_maximum_size():
    """Test tile submission at the exact size boundary."""
    room = DeadbandRoom()
    
    # Get max size from room config
    max_size = room.config.get("max_tile_size", 1024 * 1024)  # Default 1MB
    
    # Create tile at exact size
    content = "x" * (max_size - 100)  # Leave room for JSON structure
    tile = {
        "phase": "P1",
        "content": content,
        "metadata": {"size_test": True}
    }
    
    # Serialize to check actual size

    tile_json = json.dumps(tile)
    
    # Should be at or under limit

    assert result["status"] == "accepted"
```

### 5. **Test: Protocol Phase Transition with Invalid Tile Sequence**
**Scenario:** Agent submits P2 tile but room is in P0, but tile claims to be "retroactive validation".
**Edge:** Protocol enforcement vs flexible interpretation.
**Expected:** Strict protocol should reject. Flexible might allow with warning.

```python
def test_out_of_phase_tile_with_retroactive_claim():
    """Test tile claiming to validate earlier phase work."""
    room = DeadbandRoom()
    assert room.current_phase == "P0"
    
    tile = {
        "phase": "P2",
        "content": "Optimization results for P0 work",
        "metadata": {
            "agent": "test",
            "retroactive": True,
            "validates_phase": "P0"
        }
    }
    
    result = room.submit_tile(tile)
    
    # According to strict Deadband Protocol: never skip to P2
    if room.strict_protocol:
        assert result["status"] == "rejected"
        assert "P0" in result.get("error", "")  # Should mention current phase
    else:
        # Flexible implementation might allow with warning
        assert result["status"] in ["accepted", "warning"]
```
