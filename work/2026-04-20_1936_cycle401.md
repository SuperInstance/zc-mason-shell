## Edge-Case Test Scenarios
Assuming a typical room API (initialize, step, reset) and protocol logic, here are 5 edge cases:

### 1. **Test: Protocol Violation – Skip to P2**
- **Scenario**: Agent attempts to optimize (P2) before mapping negative space (P0).
- **Expected**: Room rejects tile, returns error or negative reward.
- **Check**: Ensures strict priority enforcement.

### 2. **Test: Empty Negative Space (P0)**
- **Scenario**: Agent reports no negative space (empty set).
- **Expected**: Room accepts as valid P0, transitions to P1.
- **Check**: Handles boundary condition where nothing is forbidden.

### 3. **Test: Overlapping Safe Channels (P1)**
- **Scenario**: Agent identifies safe channels that intersect with negative space.
- **Expected**: Room rejects tile (invalid P1), requires re-mapping.
- **Check**: Validates consistency between P0 and P1.

### 4. **Test: Maximum Tile Flood**
- **Scenario**: Agent submits 1000+ tiles in rapid succession.
- **Expected**: Room throttles or queues, maintains state integrity.
- **Check**: Stress tests room's ingestion and state management.

### 5. **Test: Partial Protocol Reset**
- **Scenario**: Room reset during P1, but agent tries to continue from previous P2.
- **Expected**: Room re-initializes to P0, ignores stale state.
- **Check**: Ensures clean slate on reset.

# test_deadband_room_edge.py
import pytest
from plato_torch.rooms.deadband_room import DeadbandRoom

def test_skip_to_p2_violation():
    """P0: Agent cannot jump to optimization before negative space."""
    room = DeadbandRoom()
    # Simulate agent submitting P2 tile without P0/P1
    tile = {"phase": "P2", "action": "optimize", "data": {...}}
    result = room.submit_tile(tile)
    assert result["accepted"] == False
    assert "violation" in result.get("error", "").lower()

def test_empty_negative_space():
    """P0: Empty negative space is valid."""
    room = DeadbandRoom()
    tile = {"phase": "P0", "negative_space": []}
    result = room.submit_tile(tile)
    assert result["accepted"] == True
    assert room.current_phase == "P1"

def test_overlapping_safe_channels():
    """P1: Safe channels must not intersect negative space."""
    room = DeadbandRoom()
    # First, establish P0
    room.submit_tile({"phase": "P0", "negative_space": ["rock", "wall"]})
    # Then submit P1 with overlap
    tile = {"phase": "P1", "safe_channels": ["rock", "open"]}  # "rock" is in negative space
    result = room.submit_tile(tile)
    assert result["accepted"] == False

def test_max_tile_flood():
    """Stress: Room handles high-volume tile submission."""
    room = DeadbandRoom()
    for i in range(1000):
        tile = {"phase": "P0", "negative_space": [f"hazard_{i}"]}
        result = room.submit_tile(tile)
        # Should not crash or corrupt state
        assert room.tile_count == i + 1
    assert room.tile_count == 1000

def test_reset_during_p1():
    """Reset: Room resets to P0, discarding partial progress."""
    room = DeadbandRoom()
    room.submit_tile({"phase": "P0", "negative_space": ["fire"]})
    room.submit_tile({"phase": "P1", "safe_channels": ["water"]})
    assert room.current_phase == "P2"
    room.reset()
    assert room.current_phase == "P0"
    # Attempt to submit P2 should fail
    result = room.submit_tile({"phase": "P2", "action": "optimize"})
    assert result["accepted"] == False
